  - 环境配置信息
  - 问题排查参考

## 文档维护

- 所有重要的操作命令都应该记录在相应的文档中

- 新增功能时应该同步更新相关文档

## 使用说明

1. **新开发者**: 先阅读 [setup-commands.md](setup-commands.md) 了解项目设置过程
2. **问题排查**: 参考命令记录文档中的执行结果进行对比
3. **环境重建**: 按照记录的命令步骤重新搭建开发环境
